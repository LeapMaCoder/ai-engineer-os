# 01_Research

## 职责

沉淀支撑产品决策的证据。

Research 喂给 Product；**不**授权写代码。

## 证据标注（强制）

| 标记 | 含义 |
|------|------|
| **Confirmed** | 已验证 / 可复核事实 |
| **Hypothesis** | 假设 |
| **Unknown** | 未知 |

禁止把 Hypothesis 写成确定性结论。

## 规则

- 调研笔记注明日期与来源局限。
- 区分事实、假设与建议。
- 过时调研归档，勿悄悄覆盖。
- 本阶段禁止功能设计、页面设计、技术方案与代码。
