# 00_Vision

## Purpose

Define **why LeapMa exists** and the long-term north star.

Vision is the top of the Source of Truth chain:

**Vision → Product → Specification → Architecture → Code → Test**

## Contains

- Product vision statements
- Mission / principles
- Long-term outcomes (not sprint features)
- Non-goals at the company/product level

## Does not contain

- Feature lists disguised as vision
- Tech stack choices
- Implementation details

## Entry criteria for downstream work

Product definitions must cite an active Vision document.
