# 00_Vision

## 职责

定义 **LeapMa 为什么存在**，以及长期北极星。

Vision 位于 Source of Truth 链路顶端：

**Vision → Product → Specification → Architecture → Code → Test**

## 包含

- 产品愿景陈述
- 使命 / 原则
- 长期结果（不是 Sprint 功能清单）
- 公司/产品级非目标

## 不包含

- 伪装成愿景的功能列表
- 技术栈选型
- 实现细节
