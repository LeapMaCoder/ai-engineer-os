# 假设验证台账

维护 H1–H8 的验证状态。  
每场访谈后更新「证据」；状态变更需写明依据。

## 状态枚举

| 状态 | 含义 |
|------|------|
| **Unvalidated** | 未验证（默认） |
| **Supported** | 被多条独立证据倾向支持（仍非最终真理） |
| **Refuted** | 被证据倾向证伪 |
| **Mixed** | 证据冲突，需分群解释 |
| **Unknown** | 信息不足，无法判断 |

证据级别仍可标：Confirmed / Hypothesis / Unknown（单条证据质量）。

```mermaid
flowchart LR
  U[Unvalidated] --> S[Supported]
  U --> R[Refuted]
  U --> M[Mixed]
  U --> K[Unknown]
  S --> M
  R --> M
```
